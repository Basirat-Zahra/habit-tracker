# Habit Manager CLI (C++)

A simple command-line habit tracking application built using C++.
This project was created to practice core C++ concepts and OOP design.

## Features
- Add new habits
- Mark habits as completed
- Track progress


## Concepts Practiced
- Object-Oriented Programming
- STL (std::vector)
- Encapsulation

## How to Run

Compile:
g++ main.cpp -o habitmanager

Run:
./habitmanager
